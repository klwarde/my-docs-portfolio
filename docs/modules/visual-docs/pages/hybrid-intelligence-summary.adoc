= Hybrid Intelligence (Frank van Harmelen, Knowledge Graph Insights Podcast)

== Overview

A visual summary of link:https://knowledgegraphinsights.com/frank-van-harmelen/[Frank van Harmelen's conversation on hybrid intelligence], partnerships between humans and machines grounded in shared understanding, complementary strengths, and neuro-symbolic reasoning. The episode explores Theory of Mind, collaborative AI, symbolic + neural systems, real-world use cases, and the CARE framework (Collaboration, Adaptation, Responsibility, Explainability). I sketched it because I value the idea that AI systems should work with people, and can't replace them.
