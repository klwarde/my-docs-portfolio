= The Diátaxis Framework (Talk by Daniele Procida)

== Overview

A visual breakdown of Procida's influential link:https://diataxis.fr/[Diátaxis framework] for structuring technical documentation: Tutorials, How-To Guides, Technical Reference, and Explanation. Diátaxis encompasses four modes of documentation, how they differ, and how they work together to create healthy documentation ecosystems. The framework is foundational to modern documentation practice.

For a clear and well-executed example of the framework, link:https://kubernetes.io/docs/home/[explore the Kubernetes documentation].
