= 7 Habits of Increasingly Technical Technical Writers (Talk by Janine Chen)

## Overview

A visual breakdown of link:https://www.youtube.com/watch?v=SkpXpvTk6iU[Janine Chen's inspiring talk at Write the Docs Portland 2025]. I sketched a summary of the talk because it has great advice for anyone who doesn't think they're technical enough to be a good technical writer.
