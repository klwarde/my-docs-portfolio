= AI Ethics (Vilas Dhar, LinkedIn Learning)

== Overview

A three-part visual summary of Vilas Dhar's LinkedIn Learning course link:https://www.linkedin.com/learning/ethics-in-the-age-of-generative-ai[Ethics in the Age of Generative AI] providing a practical framework needed to implement responsible AI inside organisations. This course explores responsible data practices, well-defined boundaries, transparency, training, governance, and long-term ethical stewardship. I sketched the summary to understand how ethics becomes operational, moving from abstract principles to daily decision-making.
