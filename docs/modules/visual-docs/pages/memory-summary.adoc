= The Role of Memory in Digital Branding for AI (Andrea Volpini, Knowledge Graph Insights Podcast)

## Overview

A visual exploration of link:https://knowledgegraphinsights.com/andrea-volpini/[Andrea Volpini's conversation on the Knowledge Graph Insights Podcast] about how different AI systems model "memory, and how brands can design content that aligns with semantic, episodic, and procedural memory structures. The conversation explores AI memory models, structured knowledge, vector representations, online behaviour, and the shift from keyword search to agentic retrieval. I sketched it to understand how AI agents store, retrieve, and frame information—and how this changes the future of digital communication.
