= Crafting GPTs for Productivity & Innovation  

== Overview

A three-part visual summary of Jules White's LinkedIn Learning course, link:https://www.linkedin.com/learning/chatgpt-crafting-exceptional-gpts-for-enhanced-productivity-and-innovation/gpts-and-chatgpt-introduction-to-crafting-your-own-powerful-ai[ChatGPT: Crafting Exceptional GPTs for Enhanced Productivity and Innovation]. This course explores how to design user-centred, safe, and high-performing custom GPTs. Key themes include UX foundations, boundary-setting, ambiguity handling, retrieval-augmented generation (RAG), advanced prompt patterns, and systematic testing. I created these sketchnotes to synthesise the material.
